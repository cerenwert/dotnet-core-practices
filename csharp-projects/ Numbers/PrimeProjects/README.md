# Prime Number Analyzer (C# Console Application)

## 📌 Description
This console application accepts 20 positive integers from the user and separates them into two different lists:

- Prime numbers
- Non-prime numbers

The program validates user input, sorts both lists in descending order, and displays the count and average of each group.

---

## 🎯 Features

- Accepts exactly 20 positive integers
- Prevents negative and non-numeric inputs
- Classifies numbers as prime or non-prime
- Sorts both lists in descending order
- Displays:
  - Number of elements in each list
  - Average value of each list

---

## 🧠 Concepts Used

- `ArrayList`
- Custom sorting with `IComparer`
- `Math.Sqrt()` for optimized prime checking
- Input validation with `TryParse`
- Loop structures (`for`, `foreach`)
- Method separation and clean code structure

---

## 🚀 How It Works

1. The user enters 20 positive integers.
2. Each number is checked to determine if it is prime.
3. Prime and non-prime numbers are stored in separate lists.
4. Both lists are sorted in descending order.
5. The program displays:
   - The sorted lists
   - The count of elements
   - The average of each list
