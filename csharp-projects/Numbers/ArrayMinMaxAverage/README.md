# Array Min-Max Average Application

This is a simple C# console application that:

- Accepts 20 numbers from the user
- Finds the smallest 3 numbers
- Finds the largest 3 numbers
- Calculates the average of each group
- Prints both averages and their total to the console

## How It Works

1. The program stores 20 integers in an array.
2. The array is sorted using `Array.Sort()`.
3. The first 3 elements represent the smallest numbers.
4. The last 3 elements represent the largest numbers.
5. The averages of both groups are calculated.
6. The program prints:
   - Smallest 3 numbers average
   - Largest 3 numbers average
   - Sum of both averages

## Technologies Used

- C#
