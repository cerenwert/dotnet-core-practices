# Vowel Extractor and Sorter

This is a simple C# console application that:

- Accepts a sentence from the user
- Extracts all vowel letters
- Stores them in an array
- Sorts the array alphabetically
- Prints the sorted vowels to the console

## How It Works

1. The program reads a sentence from the user.
2. It converts the sentence to lowercase to avoid case sensitivity issues.
3. It checks each character to determine whether it is a vowel.
4. All detected vowels are stored in a dynamic list.
5. The list is converted to an array.
6. The array is sorted using `Array.Sort()`.
7. The sorted vowels are printed to the console.


## Purpose

This project demonstrates:

- String manipulation
- Character iteration
- Array usage
- Sorting with `Array.Sort`
- Basic control structures
